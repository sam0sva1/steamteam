# SteamTeam
### Helps you and your friends chose a game

##### Before using
In the root directory create a file named config.js
and fill the steam key in.
```
{
    "steam_key": "12312312312312312312312312312312"
}
```

##### To start:
```sh
$ yarn
$ yarn start
```

##### For development:
- in one terminal window
    ```sh
    $ yarn
    $ yarn watch
    ```

- in another terminal window

    ```sh
    $ yarn dev
    ```

##### Note
- If you don't have some data to put it in a cache, it's better to wait until the cache warms up.
